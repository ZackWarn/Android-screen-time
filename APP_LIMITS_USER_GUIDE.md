## ğŸš€ How to Use App Limits

### Step 1: Open the App
Launch ScreenTime app on your device.

### Step 2: Navigate to Limits Tab
Tap the **"Limits"** tab (gear icon) at the bottom navigation bar.

### Step 3: Add Your First Limit
1. Tap the **+ (Plus)** button in the top right
2. Select an app from the list (e.g., Instagram, YouTube)
3. Set your daily limit in minutes (e.g., 30 minutes)
4. Tap "Save"

### Step 4: Monitor Usage
- The app will show your current usage vs. limit
- Progress bar fills as you use the app
- Green = safe zone
- Red = approaching/exceeded limit

### Step 5: Get Notified
- **5 minutes remaining**: You'll get a warning notification
- **Limit exceeded**: App will be force-closed and blocked

### Step 6: View Block Screen
When an app is blocked, you'll see:
- Large warning icon
- Usage statistics (e.g., "20/20 min used")
- Message: "This app is blocked for today"
- "Go to Home" button

## âš ï¸ Troubleshooting

### App Not Blocking
**Solution:**
1. Check if limit is enabled (toggle ON)
2. Verify Usage Access permission granted
3. Restart app
4. Check if service is running (notification should be visible)

### Service Stopped
**Solution:**
1. Open ScreenTime app (auto-starts service)
2. Disable battery optimization (see Permissions section)
3. Make sure "Don't optimize" is selected

### Notifications Not Showing
**Solution:**
1. Grant notification permission
2. Check notification settings in Android
3. Ensure "Usage Access" permission is granted

### Wrong Usage Numbers
**Solution:**
1. Wait a few minutes (updates every 5 seconds)
2. Kill and restart the app
3. Check if multiple users on device (usage is per-user)
